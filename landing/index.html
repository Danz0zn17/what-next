How it works

Drop it in once.
It travels everywhere.

Architecture

Built on three principles

Always-on sync

SQLite caches locally on your machine. A Postgres instance on Railway handles cloud storage. Every write goes through both. Works offline; catches up automatically.

AES-256-GCM encryption

Memory is encrypted per-user at rest using AES-256-GCM. The cloud stores ciphertext. Your key never leaves your machine. We cannot read your data.

Open protocol

Built on MCP (Model Context Protocol). No proprietary plugins. Any MCP-compatible surface you add in future gets full memory without extra configuration.

Setup

Two minutes to add it anywhere

Add What Next to your MCP config file. That's it. No plugins, no browser extensions, no accounts per-surface.

claude_desktop_config.json
{
  "mcpServers": {
    "what-next": {
      "command": "node",
      "args": ["src/server.js"],
      "env": {
        "WHATNEXT_API_KEY": "your-key"
      }
    }
  }
}
